## Mục tiêu

Bot đề xuất strategy → export sang Pine v6 để user paste vào TradingView,
test real-time trên data nhiều timeframe hơn, không phụ thuộc vnstock.

## 5 strategy template hỗ trợ

### T1. Breakout (20-day Donchian)
```pine
//@version=6
strategy("Breakout 20d", overlay=true, initial_capital=100000000)

length = input.int(20, "Lookback")
vol_mult = input.float(2.0, "Volume multiplier")

hh = ta.highest(high, length)[1]
vol_ma = ta.sma(volume, length)
entry_cond = close > hh and volume >= vol_ma * vol_mult

stop = close * 0.92   // -8%

if entry_cond
    strategy.entry("Long", strategy.long)
    strategy.exit("Exit", from_entry="Long", stop=stop, profit=hh * 1.15)
```

### T2. MA Cross (Golden/Dead cross)
```pine
//@version=6
strategy("MA Cross 20/50", overlay=true)

ma_fast = ta.sma(close, input.int(20))
ma_slow = ta.sma(close, input.int(50))

long_cond  = ta.crossover(ma_fast, ma_slow)
exit_cond  = ta.crossunder(ma_fast, ma_slow)

if long_cond
    strategy.entry("Long", strategy.long)
if exit_cond
    strategy.close("Long")
```

### T3. RSI Mean-Reversion
```pine
//@version=6
strategy("RSI Mean Reversion", overlay=false)

rsi_len = input.int(14)
oversold = input.int(30)
overbought = input.int(70)
trend_ma = ta.sma(close, 50)

rsi_v = ta.rsi(close, rsi_len)
long_cond = rsi_v <= oversold and close > trend_ma
exit_cond = rsi_v >= overbought

if long_cond
    strategy.entry("Long", strategy.long)
if exit_cond
    strategy.close("Long")

plot(rsi_v)
hline(oversold, "OS")
hline(overbought, "OB")
```

### T4. Ichimoku Cloud
```pine
//@version=6
strategy("Ichimoku", overlay=true)

tenkan = (ta.highest(high, 9) + ta.lowest(low, 9)) / 2
kijun  = (ta.highest(high, 26) + ta.lowest(low, 26)) / 2
span_a = (tenkan + kijun) / 2
span_b = (ta.highest(high, 52) + ta.lowest(low, 52)) / 2

above_cloud = close > span_a and close > span_b
golden_cross = ta.crossover(tenkan, kijun)

if golden_cross and above_cloud
    strategy.entry("Long", strategy.long)
if close < kijun
    strategy.close("Long")
```

### T5. Multi-Factor Screen (entry only)
```pine
//@version=6
// Note: multi-factor cần cross-section — Pine không native.
// Template này chỉ confirm 1 ticker đạt tiêu chí factor.
indicator("MF Screen", overlay=false)

momentum_20 = close / close[20] - 1
rsi_14 = ta.rsi(close, 14)
vol_surge = volume > ta.sma(volume, 20) * 1.5

meets = momentum_20 > 0.05 and rsi_14 > 55 and rsi_14 < 75 and vol_surge
plotshape(meets, style=shape.triangleup, color=color.green)
```

## Rules generate

### R1. Template selection
- User yêu cầu `breakout` / `ma_cross` / `rsi_mr` / `ichimoku` / `mf_screen`
  → pick template tương ứng.
- Substitute parameters (length, threshold) từ context.

### R2. Customization
- VN fee: commission 0.25% round-trip default.
- Initial capital 100,000,000 VND (khớp bot).
- `overlay=true` cho price-based, `overlay=false` cho indicator-based.

### R3. Comment header
```pine
// Generated by vnstock-bot {date}
// Strategy: <name>
// Source: skills/{skill_name}
// Parameters: <parameters>
// Hướng dẫn: paste vào TradingView Pine Editor → "Add to Chart"
```

### R4. Export file
- Save ra `data/export/pine/<strategy_name>_<date>.pine`.
- Telegram reply: link file + preview 10 dòng đầu.
